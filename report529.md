## git のルール

### fetch

- fetch とはリモートリポジトリから最新情報をローカルリポジトリに持ってくるコマンドです。
- fetch をしてもファイルは更新されない
- ローカルリポジトリが更新されるだけ

### marge

- 各ブランチで編集した作業を反映させるためのコマンド

  #### marge の注意点

  - commit が済んでいないファイルが残っている状態で　 git marge しない
  - commit が済んでいないファイルが残っている状態で　 git checkout しない

### pull

- marge と fetch が混じったもの
